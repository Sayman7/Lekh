# How To Run

### NOTE
- You should have "cmake" installed

## Run
- You can use to methods to run this

#### Method 1
- Give execution permission to "run-build.sh"
```bash
chmod +x run-build.sh
```

- Execute the file "run-build.sh"
```bash
./run-build.sh
```

#### Method 2
- Run script using bash cammand
```bash
bash ./run-build.sh
```

# File Structure
This is how files will be structured for this project.

```
/Lekh
├── CMakeLists.txt       # Top-level CMake configuration file
├── src                  # Source files
│   ├── main.cpp         # Main application file
│   └── Class.cpp        # Implementation of Class
├── include              # Header files
│   └── Class.hpp        # Declaration of Class
├── tests                # Unit tests
│   ├── CMakeLists.txt   # CMake configuration for tests
│   └── tests.cpp        # Test implementation
├── build                # Build directory (created by CMake)
├── docs                 # Documentation files
│   └── ex-doc.md        # Example documentation
├── README.md            # Project README.md file
├── file-structure.md    # File structure of project
├── .git                 # Git directory
├── .gitignore           # Git igrnore file
└── run-build.sh         # To run the project

```
